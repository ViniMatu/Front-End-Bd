Sobre

Este projeto foi desenvolvido para a disciplina de Aspectos em Banco de Dados e tem como objetivo aplicar os conceitos estudados por meio da construção de uma aplicação real.


A proposta consiste em uma interface que permite visualizar comentários e avaliações de clientes sobre filmes.


Para a implementação, escolhemos utilizar o Amazon DynamoDB como Sistema de Gerenciamento de Banco de Dados (SGBD), explorando suas características NoSQL, como alta escalabilidade, baixa latência e estrutura orientada a documentos.


A escolha do DynamoDB também permitiu o uso de chaves de partição e sort keys para organizar os dados de forma eficiente, além de integrar a aplicação com uma arquitetura moderna e baseada em serviços.
